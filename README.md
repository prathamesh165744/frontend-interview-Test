# CA Monk Blog Application

Frontend blog application built as part of CA Monk Frontend Developer Intern assignment.

## Tech Stack
- React + TypeScript
- TanStack Query
- Tailwind CSS
- shadcn/ui
- JSON Server

## Features
- View all blogs
- View blog details
- Create a new blog
- Responsive UI
- Proper loading & error states

## Setup Instructions

```bash
npm install
npm run server
npm run dev
